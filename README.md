# Tik Downloader

Download TikTok videos without watermark and in HD quality directly from TikTok.

Tik Downloader is a modern UserScript that integrates directly into TikTok's interface, allowing you to download videos with a single click.

> **⭐ Recommended Installation:** `tikdownloader.client.user.js`
>
> The Client version automatically keeps Tik Downloader updated with the latest releases from GitHub.

# 🇺🇸 English

## ✨ Features

- Download TikTok videos without watermark
- HD video support
- One-click download directly from TikTok
- Real-time progress bar
- Smart file naming
- Automatic updates through the Client version

## 📦 Project Structure

```text
📦 Tik Downloader
├── tikdownloader.client.user.js
└── tikdownloader.user.js
```

## 📊 Client vs Standalone

| Feature | Client | Standalone |
|----------|---------|---------|
| Download without watermark | ✅ | ✅ |
| HD downloads | ✅ | ✅ |
| Automatic updates | ✅ | ❌ |
| Local cache | ✅ | ❌ |

## ⚙️ Technologies

- JavaScript (ES6+)
- Tampermonkey
- TikWM API
- GM_xmlhttpRequest
- GM_download
- GM_setValue
- GM_getValue

## 🚀 Installation

1. Install Tampermonkey.
2. Install `tikdownloader.client.user.js` (recommended).
3. Open TikTok.
4. Open a video page.
5. Click the Tik Downloader button.

## 🔄 Automatic Updates

The Client version automatically checks GitHub for new releases, downloads updates, stores them locally, and keeps the downloader up to date.

## 🔒 Privacy

Tik Downloader does not collect, store, or share personal information.
